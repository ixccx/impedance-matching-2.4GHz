# Impedance Matching Network Design at 2.4 GHz

**King Abdulaziz University | EE423 | 2025**

## Overview
Designed and simulated three RF impedance matching networks 
for a load of ZL = 22−j37 Ω matched to a 50-Ω system at 2.4 GHz.

## Methods Implemented
- L-Network (Lumped Components)
- Open-Shunt Stub
- Short-Shunt Stub

## Results
| Method | S11 (dB) | Bandwidth |
|--------|----------|-----------|
| L-Network | -65.5 dB | 710 MHz |
| Short Stub | -12.8 dB | 290 MHz |
| Open Stub | -12.7 dB | 430 MHz |

## Tools
- Qucs (Circuit Simulation)
- Smith Chart Analysis
